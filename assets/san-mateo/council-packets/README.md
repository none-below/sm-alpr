These text files are machine-extracted from San Mateo City Council meeting
packet PDFs obtained via the PrimeGov public API. They are provided for
search and analysis purposes only.

They are not original documents. Text extraction may introduce errors,
especially in scanned/image-based pages where OCR is used. All claims
derived from these files should be verified against the original source
packets.

Original packets can be accessed at:
  https://sanmateo.primegov.com/Public/CompiledDocument?meetingTemplateId={id}&compileOutputType=1

The meetingTemplateId for each packet is recorded in its metadata.json file.
