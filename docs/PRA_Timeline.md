## CPRA deadline legend

- **10-day determination** (Gov. Code § 7922.535(a)): agency must notify requester within 10 calendar days whether it will produce records.
- **+14-day extension** (Gov. Code § 7922.535(b)): "unusual circumstances" can extend by up to 14 more days via written notice (24-day total).
- **Reasonable time for production** (Gov. Code § 7922.500, § 7922.530(b)): after determination, records must be produced "promptly"; rolling production is permitted.
- Agency-promised dates (e.g. "updates no later than X") are the more actionable deadline after determination — listed in each entry below.
