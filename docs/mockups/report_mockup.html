California ALPR Law — Key Requirements

CA Civil Code §§1798.90.51–.55 (SB 34) governs ALPR use by California agencies:

  • Sharing restricted to public agencies (§.55(b)): ALPR data may only be shared with public agencies. Private universities, HOAs, and vendors in a sharing list likely violate this provision.
  • Usage and privacy policy required (§.51(a)): Operators must conspicuously post a policy covering uses, access, retention, auditing, and accountability.
  • Audit requirements (§.51(b)(5)): Policies must include provisions for auditing access.
  • End-user obligations (§.53): Agencies accessing ALPR data must maintain a usage policy, even if they don't operate cameras.

Common compliance gaps observed across California agencies:

  • Private entities (private universities, HOAs) appearing in sharing lists
  • No downloadable audit CSV despite §.51(b)(5) audit requirement (~91% of CA agencies)
  • No published search audit policy (~64% of CA agencies)
  • Vendor contracts granting independent disclosure authority (e.g., Flock MSA §5.3)
  • No user offboarding procedures when personnel leave
