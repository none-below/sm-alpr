California ALPR Law — Key Requirements

CA Civil Code §§1798.90.51–.55 (SB 34) requires California ALPR operators to:

  • Maintain and conspicuously post a usage and privacy policy (§.51(a)) covering authorized uses, data access, retention, auditing, and accountability.
  • Share only with public agencies (§.55(b)).
  • Implement audit provisions (§.51(b)(5)) for access to ALPR information.

A public transparency page is the most visible way agencies demonstrate this compliance. 132 California agencies currently publish one through Flock Safety's transparency program.
